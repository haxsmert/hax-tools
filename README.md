# hax-tools

个人工具集合，按平台分类存放。

## 目录结构

```
hax-tools/
├── agent-tools/           # AI Agent 相关工具
│   └── hermes-tools/      # Hermes Agent 辅助工具
├── windows-tools/         # Windows 平台工具
│   └── lol-check/         # LOL 游戏服检测脚本
├── mac-tools/             # macOS 平台工具
└── README.md
```

## 工具列表

### Windows

- **[lol-check](windows-tools/lol-check/)** — 通过 netstat 分析 UDP/TCP 连接识别 LOL 游戏服务器

### Agent

- **[hermes-tools](agent-tools/hermes-tools/)** — Hermes Agent 辅助工具
